# My Personal Dotfiles

This repository contains my personal dotfiles.

## Disclaimer

These configurations are highly specific to my personal setup, tools, and workflow on macOS. They are not intended to be a general-purpose template or to be cloned by others.

**Please do not clone or use this repository directly.** You are welcome to browse the files for inspiration, but attempting to use these dotfiles on your own machine will likely cause issues or overwrite your existing configurations.
